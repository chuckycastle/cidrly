# cidrly Roadmap

Development roadmap is managed through GitHub. See:

- **[Milestones](https://github.com/chuckycastle/cidrly/milestones)** - Planned releases and feature grouping
- **[Issues](https://github.com/chuckycastle/cidrly/issues)** - Feature requests and bug tracking
- **[Releases](https://github.com/chuckycastle/cidrly/releases)** - Release history and notes
- **[CHANGELOG.md](CHANGELOG.md)** - Detailed changelog with version history

## Release Strategy

cidrly follows semantic versioning with a structured release progression:

- **v0.3.1-v0.3.3**: Polish v0.3.0 foundation (safety → testing → documentation)
- **v0.4.0**: Add complex network design capabilities
- **v0.5.0**: Enable programmatic access and third-party integration

Each milestone includes a description explaining the focus and rationale for that release.
